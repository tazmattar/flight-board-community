# VATSIM Scandinavia Stand Data — Attribution

Stand coordinate data for Scandinavian airports is sourced from the [VATSIM Scandinavia](https://vatsim-scandinavia.org) Ground Radar Plugin (GRPlugin) sector files, used under a non-commercial attribution licence approved by **Thor Høgås** on behalf of VATSIM Scandinavia.

## Licence terms

1. **Non-commercial use only** — FlightBoard is non-commercial.
2. **Attribution required** — the VATSIM Scandinavia logo is displayed in the app footer whenever a Scandinavian airport is active, linking to `https://vatsim-scandinavia.org`.
3. **Upstream contributions** — coordinate corrections should be submitted back via pull requests to the relevant VATSIM Scandinavia sector file repository.

## Airports covered

732 stands across 32 airports (EKDK/ENOR/ESAA divisions):

**Denmark (EKDK):** EKCH, EKBI, EKYT, EKAH, EKRN  
**Norway (ENOR):** ENGM, ENBR, ENVA, ENZV, ENTO  
**Sweden (ESAA):** ESSA, ESGG, ESSB, ESPA, ESKN, ESOW, ESSV, ESMS, and others

## Submitting coordinate corrections

If you spot an incorrect stand position, submit a correction to the relevant VATSIM Scandinavia sector file repository on GitHub. This keeps the upstream data accurate for all tools that use it.
